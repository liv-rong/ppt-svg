# 04-示意图-元素管理与操作模块

```
┌─────────────────────────────┐
│         SVG 画布区          │
│ ┌─────┐   ┌─────┐           │
│ │文本 │   │图形 │   ...     │
│ └─────┘   └─────┘           │
│                             │
│ (可选中、拖拽、缩放、旋转等) │
└─────────────────────────────┘
```

- 支持添加文本、图形、图片等元素。
- 支持元素的选中、拖拽、缩放、旋转、删除、复制、粘贴、层级调整。
- 支持通过 Markdown 自动生成初步内容后再编辑。
